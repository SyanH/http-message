## Mix Http Message

基于 PSR-7 / PSR-17 标准的 Http 消息库，支持 Swoole

## Usage

安装：

```
composer require mix/http-message
```

文档：

- 请求：https://www.kancloud.cn/onanying/mixphp2-1/1255264
- 响应：https://www.kancloud.cn/onanying/mixphp2-1/1255265
